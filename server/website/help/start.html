Getting started

Introduction

PSI-Toolkit is a set of linguistic tools for processing texts written in various formats and various languages. The largest number of tools has been designed for Polish, but also tools for English, French, German or Russian are available.

The full strength of PSI-Toolkit is offered with command-line version, but most of the features you can test via this web-service.

What is going on here?

So all you need to process your data is to specify what you want to do, write input text and run PSI-Toolkit.

Commands

With commands you specify what you want to do (e.g. tokenize and lemmatize) or, more general, what tool (called processor) should be used to process your data (e.g. tokenizer or lemmatizer). Notice, that all data have to be firstly readed to be writed after processing. So there are three types of processors: readers, annotators and writers.
